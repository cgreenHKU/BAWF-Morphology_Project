BAWF Morphological Database

Interactive view of the morphology and frequency dataset (draft).
You can sort columns, filter, search, and scroll through the rows directly in the embedded Excel view.
Features available:
• Click column headers to sort (ascending / descending)
• Use search/filter controls (if visible in header row)
• Scroll through the table
• Download the full Excel file using the button in the top-right corner

Note: This is still a web-based view. For offline work or advanced editing, download the file.
